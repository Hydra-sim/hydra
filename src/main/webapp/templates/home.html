Welcome to HYDRA

HYDRA is a web-based simulation tool designed to analyse and simulate passengerflow at airports using a custom simulation algorithm. The solution consists in large of three different parts which are simulations, timetables and locations, The main purpose for HYDRA is to simulate passengerflow at the depature terminal of airports. The tool is developed alongside and for Avinor.

Simulations

The simulation aspect of the application is it's core functionality. The aim is to give the user an easy and user-friendly way to create and manage simulations. We have created a really visual and easy way to create simulations based on locations gusing drag and drop and user friendly user input.

Timetables

One of the crucial parts of setting up a simulation is the timetables which is a fully editable list of different train and bus routes bringing people to your location. The user can easily create, upload or edit exsiting timetables.

Locations

The third part of the solution is locations which are pre-defined simulations that are ready to run as is or, can be edited as you choose.
