Simulations

In the simulations tab you can see a list of all previous simulation. Click on simulation name to view simulation results. You can also delete, edit, share and set password on the simulations .

Timetables

In the timetables tab you can see a list of all timetables . Click on timetable name to edit timetable. You can also create new timetables and delete them.

Locations

In the presets tab you can see a list of all locations. A location is a pre-defined simulation that you can run as is or edit to customize it. You can edit, delete or create a new location.

SETUP

Right click on the screen to add a new item. A popup for entering information for the unit/units will appear and you enter it and pres add then the unit will be added. To create a direction for the passengers to go, you click shift + drag from an item to another item. You will then be able to enter in how many percent of the passengeres that should walk in that direction.

Short commands:

Drag a node to move it. Scroll too zoom working area, drag working area too move around. Click on a node or edge to select it. Click shift+backspace to delete selected edge / selected node and all connected edges. Shift+drag from one node to another to add an edge.
